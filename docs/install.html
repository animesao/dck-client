Installation

Installing dck Runtime

Linux (x86_64 / arm64)

# Download the latest binary
curl -fsSL https://github.com/anomalyco/dck/releases/latest/download/dck-linux-amd64 -o /usr/local/bin/dck

# For ARM64:
# curl -fsSL https://github.com/anomalyco/dck/releases/latest/download/dck-linux-arm64 -o /usr/local/bin/dck

chmod +x /usr/local/bin/dck

# Verify installation
dck version

From Source

# Prerequisites: Go 1.22+
git clone https://github.com/anomalyco/dck.git
cd dck
go build -o dck .
sudo cp dck /usr/local/bin/

Installing dck Panel (Web UI)

One-Command Install (Ubuntu/Debian)

curl -sSL https://raw.githubusercontent.com/animesao/dck-client/main/install.sh | sudo bash

# Custom port (if 443 is blocked):
curl -sSL https://raw.githubusercontent.com/animesao/dck-client/main/install.sh | sudo bash -s 8443

The install script will:

  • Install system dependencies
  • Install dck runtime (if missing)
  • Install Node.js and Go
  • Clone and build the panel
  • Generate self-signed TLS certificate
  • Create systemd service
  • Open firewall port

Manual Installation

# 1. Install dck
curl -fsSL https://github.com/anomalyco/dck/releases/latest/download/dck-linux-amd64 -o /usr/local/bin/dck
chmod +x /usr/local/bin/dck

# 2. Build the panel
git clone https://github.com/animesao/dck-client.git
cd dck-client
chmod +x build.sh
./build.sh
sudo cp server/dck-panel /usr/local/bin/

# 3. Run
sudo /usr/local/bin/dck-panel --port 443

Systemd Service

cat > /etc/systemd/system/dck-panel.service << 'EOF'
[Unit]
Description=dck Panel
After=network.target

[Service]
Type=simple
ExecStart=/usr/local/bin/dck-panel --port 443
Environment=DCK_HOME=/root/.dck
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
EOF

systemctl daemon-reload
systemctl enable --now dck-panel
Default login: admin / admin — change immediately after first login!
