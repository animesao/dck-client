dck Container Runtime

Lightweight, secure, and easy-to-use container runtime with a modern web panel for managing containers, files, backups, and system resources.

Get Started View Commands

Why dck?

⚡

Lightweight

Single binary, no dependencies, minimal resource overhead. Written in Go for maximum performance.

🔒

Secure

Rootless by design, seccomp profiles, capabilities dropping, and resource isolation via cgroups.

🌐

Web Panel

Full-featured Pterodactyl-like interface with file manager, console, backups, and user management.

📦

Game Servers

Pre-configured presets for Minecraft, Rust, SRCDS, and more. Perfect for game hosting.

🔄

OverlayFS

Efficient container storage using overlay filesystem with instant snapshots and low disk usage.

🔧

REST API

dck-wings provides a full REST API for remote management with WebSocket console support.

Quick Start

# Install dck (Ubuntu/Debian)
curl -fsSL https://github.com/anomalyco/dck/releases/latest/download/dck-linux-amd64 -o /usr/local/bin/dck
chmod +x /usr/local/bin/dck

# Create and start a container
dck create nginx:latest --name myweb
dck start myweb

# List running containers
dck ps
Full documentation and source code available on GitHub. Web panel source: dck-client.
